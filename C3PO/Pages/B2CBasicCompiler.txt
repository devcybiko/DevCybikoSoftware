                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">Music (foreground/background)</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">Simple Dialog Boxes</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">Fonts</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">Inline C calls</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">Menus</FONT></B></LI>

                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">Vibrate</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">Fixed-point variables (decimals)</FONT></B></LI>

                                                            <P><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">Release 5 is perhaps the most significant release of B2C - and is also the last. 
                                                                 These new features are added in v5:</FONT></B></P>
                                                            <UL>
                                                                <LI><B><FONT SIZE="-2" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif"> </FONT><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">All documentation now stored in 
                                                                    B2C.chm file (windows html help)</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif"> Double variables are implemented.  Actually implemented as 32-bit fixed point math.  16 bits of integer and 16 bits of decimal.  Gives a range of –32767.0000 to +32767.0000 (four digits of accuracy).</FONT>
                                                                    </B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">+, -, *, /</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">sin, cos, tan, sqrt, inv – functions</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">all DIM statements MUST include an AS clause</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">inputxy – positions the cursor and allows input at that point</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">menuxy – positions the menu </FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">OPTION MULTITASKING OFF – when the app loses focus it normally will continue to run. Turning multitasking 
                                                                    off will prevent that</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">Help key processing works</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">OPTION HELP OFF – turns off help key processing</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">OPTION 3DSPRITES – set the maximum number of 3d sprites</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">OPTION 3DROOMS – set the maximum number of 3d rooms</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">wrap – displays text in a word-wrapping text box</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">rename – renames a file</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">remove – deletes a file</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">exists() – determines if a file exists</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">load – copies a file from the app's archive to the flash</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">len() – returns the length of a string</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">menuxy – positions a menu on the screen</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">rect – draws an empty rectangle</FONT></B></LI>
                                                                <LI><B><FONT SIZE="-1" FACE="Arial,Helvetica,Univers,Zurich BT,sans-serif">rectfill – draws a filled rectangle</FONT></B></LI>
