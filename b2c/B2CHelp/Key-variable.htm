Topic : Key-variable

Language Level : CyBasic-1

Effectivity : B2C v2

Syntax :

Key

Description :

Key returns the value of the last depressed key. This function contends with the Escape key processing and it is recommended that the programmer specify OPTION ESCAPE OFF. In this case the programmer should check for the ESC key. It is also worthy of note that the Key variable takes 1/10 of a second to operate thereby slowing most programs down. It is recommended that the programmer use the Key Function for most keyboard sampling as it is virtually instantaneous in processing.

Use the #defined values or numbers from the table below

#KEY_A - #KEY_Z

#KEY_0 - #KEY_9

#KEY_ENTER, #KEY_SELECT, #KEY_TAB, #KEY_INS, #KEY_DEL

#KEY_UP, #KEY_DOWN, #KEY_LEFT, #KEY_RIGHT
