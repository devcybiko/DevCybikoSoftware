Topic : Sprite_Get

Language Level : B2C Extension

Effectivity : B2C v3d

Syntax :

Sprite_get n, x, y, z, mode, w, h

Description :

Sprite_get will retrieve the position (x,y), bitmap number (z), FLIP value (mode), and width and height (w,h).

Example

Dim x as int

Dim y as int

Dim z as int

Dim mode as int

Dim w as int

Dim h as int

Sprite_get 1, x,y,z,mode,w,h
