Topic : ToCyid

Language Level : B2C Extension

Effectivity : B2C v3d

Syntax :

Cyid = ToCyid(username)

Description :

ToCyid will convert a username (nickname) into a numeric CyID. CyId must be a long integer

Example :

cyid = ToCyid(username)
