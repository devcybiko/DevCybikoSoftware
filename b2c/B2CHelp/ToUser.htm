Topic : ToUser

Language Level : B2C Extension

Effectivity : B2C v3d

Syntax :

name = ToUser(cyid)

Description :

ToUser will convert a numeric CyID into a username (nickname). CyID must be a Long Integer.

Example :

cyid = ToCyid(username)

See Also:

SendMessage, OnMessage, UserMenu, toCyid, Tokenize
