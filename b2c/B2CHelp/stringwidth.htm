Topic : Stringwidth()

Language Level : B2C Extension

Effectivity : B2C v5

Syntax :

n = stringwidth(string)

Description :

Stringwidth returns the number of x-pixels the ‘string’ will take if displayed in the current font.

Example :

OPTION C_COORDS

Dim x as int

Dim y as int

Dim s[32] as char

Input "enter your name", s

x = stringwidth(s)

y = stringheight(s)

‘ center text

cls

printxy (160-x)/2, (100-y)/2, s
