Topic : Right

Language Level : CyBasic-2

Effectivity : B2C v2

Syntax :

Right destination, source, length

Description :

Right will copy into the destination string from the source string for length characters.

Example :

Dim name[32] as char

Dim last5[10] as char

Input "Name", name

Right last5, name, 5

Print "Last part=", last5
