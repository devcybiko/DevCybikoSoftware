Topic : Circfill

Language Level : B2C Extension

Effectivity : B2C v5

Syntax :

Circfill X,Y,R

Description :

Circfill will draw a filled circle at location X, Y with a radius of R.

Example :

OPTION SHOW OFF

OPTION ESCAPE OFF

dim r as int

sub draw_circle(r as int)

cls

circfill 0,0,r

redraw show

end sub

draw_circle(r)

while true

if key(#KEY_LEFT) then

r=r+1

draw_circle(r)

endif

if key(#KEY_RIGHT) then

r=r-1

draw_circle(r)

endif

if key(#KEY_SPACE) then exit while

wend
