Topic : Key-function

Language Level : B2C Extension

Effectivity : B2C v2c

Syntax :

Key(keynumber)

Description :

Key(keynumber) returns TRUE (nonzero) if keynumber is depressed and FALSE (0) otherwise. See Key-variable, for a chart of key name and key numbers. This feature was added in Version 2c to facilitate faster keyboard checking. "Key" (the key-variable) is kept for backwards compatibility - but takes 1/10 second per call. Key() (the function) is essentially instantaneous.

It is impossible to detect the ESC key with this function. This is due to a limitation in the Cybiko DirectKeyboard function. It is recommended that the programmer use OPTION ESCAPE OFF to turn off escape key processing and use the key function to detect key presses. Also, the programmer will then have to detect some other key for exiting the program - such as the KEY_SPACE key. See the Key variable for a list of key values.

See the OPTION ESCAPE function for limitations of escape key processing.

Example :

OPTION ESCAPE OFF

Dim x as int

Dim y as int

Dim z as int

Sprite 1, "lemming.pic"

X=0

Y=0

Z=0

While 1

If key(#KEY_LEFT) then x=x-1

If key(#KEY_RIGHT) then x=x+1

If key(#KEY_SPACE) then exit program

Move x,y,z

Redraw

wend
