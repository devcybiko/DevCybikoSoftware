Topic : OPTION-ESCAPE

Language Level : B2C Extension

Effectivity : B2C v3a

Syntax :

OPTION ESCAPE ON (default)

OPTION ESCAPE OFF

Description :

The OPTION ESCAPE function enables and disables the escape key checking that occurs inside a for or while loop. Escape key checking can interfere with normal Key variable calls by swallowing key events. Also each call to the escape key function (_escape(0)) takes 1/10 of a second which will slow down most programs. For interactive games OPTION ESCAPE OFF is recommended. The default of OPTION ESCAPE ON is kept in place to make B2C consistent with CyBasic.

Example :

OPTION ESCAPE OFF
