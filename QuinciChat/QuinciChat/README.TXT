Welcome to QuinciChat - the ICQ Client for Cybiko

Read the file QuinciChat.doc in the docs directory.
There are also .rtf and .txt versions. 

In case you don't already know, this program
requires 2 cybikos (one must be a classic, 
the other can be a classic or xtreme with 
a B2C password), and a PC connected to
the internet.

Enjoy!
